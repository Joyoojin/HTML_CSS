디지털 시계 만들기


원래 페이지로 돌아감
<h3 style="color: salmon;"> 디지털 시계 만들기 </h3> <div id="digClock"></div> <script> function digClock() { var today = new Date(); var day = today.getMonth() + 1 + "월 " + today.getDate() + "일 "; var time = today.getHours() + "시 " + today.getMinutes() + "분 " + today.getSeconds() + "초"; document.getElementById("digClock").innerHTML = day + time; setTimeout('digClock()', 1000); } digClock(); /*출력결과 11월 30일 12시 41분 19초 : 실시간으로 시간이 초단위로 변하면서 출력*/ </script>
