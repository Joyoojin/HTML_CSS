pwd 현재 위치 출력
~:현재 로그온한 홈폴더
 . : 현재폴더
.. : 상위폴더
clear : 화면지움(리눅스) // (cls : CMD명령프롬포트)
/ : 최상위경로
ls : 폴더의 내용출력 (리눅스 명령어)//(dis : windows CMD)
	ls 출력될때 뒤에 /붙은건 폴더, 아니면 파일
ls -la : 자세히 출력
탭 키 : 자동완성

		git이 사용할 수 있도록 초기화 
$ git init  : git 초기화
	.git 폴더가생성됨 : git의 설정파일이 저장됨

폴더명이나 파일이름 앞에 .을 찍으면 숨김폴더,파일이되어 ls-la로 봐야 보임
 ls  -a : 숨어있는 파일도 출력하어
ls -la : 숨어있는 파일까지 자세히
ls -l  : 자세히 출력
